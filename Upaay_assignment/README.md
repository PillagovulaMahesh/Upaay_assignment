# Creative Upaay Dashboard

A **Kanban-style task management dashboard** built in **React.js** with **Tailwind CSS** and **Redux**.  
This project is a front-end assignment for Creative Upaay, replicating a Figma design with functional task management features.

---

## Features

- **Pixel-perfect UI**: Matches the Figma design closely.
- **Task Management**:
  - Add tasks with title and description.
  - Move tasks between "To Do", "In Progress", and "Done".
  - Mark tasks as completed.
- **Filtering**:
  - Filter tasks based on status.
- **State Management**:
  - Uses **Redux** for centralized state.
  - **Local Storage** persistence ensures tasks remain after refresh.
- **Optional Enhancements**:
  - Can be extended with drag-and-drop task movement.

---

## Technologies Used

- **React.js** – Component-based UI framework.
- **Tailwind CSS** – Utility-first CSS for rapid UI development.
- **Redux Toolkit** – State management.
- **Local Storage** – Persistent state.
- **PostCSS & Autoprefixer** – Tailwind setup and CSS optimizations.

## Getting Started

### Prerequisites

- Node.js (v18+) and npm installed.

### Installation

1. Clone the repository:

git clone <your-repo-url>
cd creative-upaay-dashboard

2. Install dependencies:

npm install

3. Start the development server:

npm start

4. Open your browser at http://localhost:3000
